РЪфРЪф Web Fundamentals Рђћ Term Dictionary РЪФРЪФ

This repository contains a **simple educational presentation** that I use at the beginning of web development courses.

The goal is not to start coding immediately,  
but to make sure students **understand the words** they will hear every day.

The idea is straightforward:  
 terms first  
 practice later

---

РЪфРЪф Purpose of the project РЪФРЪФ

This is **not a coding tutorial**.

It exists to:
- explain common web and tech terms in plain language
- give beginners a clear mental map of the web
- reduce confusion before moving to hands-on HTML practice

In class, this works as a **reference and discussion tool**, not something to read word by word.

---

РЪфРЪф Who this is for РЪФРЪФ

This project is useful if you:

- are starting frontend development
- are new to tech in general
- often hear words like *request*, *server*, or *HTTP* and want clarity
- teach beginners and want a structured introduction

---

Each term comes with:
